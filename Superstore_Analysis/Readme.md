# Superstore Sales Analysis

## Project Overview
This project analyzes Superstore sales data using Python and Power BI to identify sales trends, profit drivers, regional performance, and product-level insights.

## Objectives
- Analyze sales and profit performance
- Identify best and worst performing products
- Understand customer segments and regional trends

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Power BI
- Excel

## Dataset
The dataset contains:
- Orders
- Sales
- Profit
- Discount
- Category & Sub-Category
- Region & Segment

## Python Analysis
Performed:
- Descriptive Statistics
- Sales & Profit Analysis
- Correlation Analysis
- Distribution Analysis
- Outlier Detection
- Trend Analysis

## Key Insights
- Technology generated the highest sales and profit.
- East region recorded the highest sales.
- Consumer segment contributed the most profit.
- Furniture showed high sales but lower profitability.
- Sales and profit peaked during the last quarter of the year.

## Recommendations
- Focus on high-performing Technology products.
- Improve profitability of Furniture products.
- Reduce losses from underperforming products.
- Increase marketing during peak sales months.
- Optimize discount strategies.
